# Behavioral-Financial-Analytics-Fraud-Intelligence-Platform
End-to-end analytics pipeline for detecting anomalous transaction patterns and segmenting customer behavior using unsupervised ML. Built with PySpark, Snowflake, Isolation Forest, and Power BI.
